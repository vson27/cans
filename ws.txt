Sequence:
Client → Server : SYN
Server → Client : SYN-ACK
Client → Server : ACK
In Wireshark you can search:
tcp.flags.syn == 1

Basic Host Discovery (Ping Scan)
nmap -sn 192.168.1.20

Basic Port Scan Find open ports.
nmap 192.168.1.20

Scan All Ports
Default scan checks 1000 ports only. To scan all ports:
nmap -p- 192.168.1.20 

Service Version Detection
nmap -sV 192.168.1.20 

Aggressive Scan
nmap -A 192.168.1.20

OS Detection
sudo nmap -O 192.168.1.20 

Scan Specific Ports Example scanning web ports:
nmap -p 80,443,8080 192.168.1.20 

Run Nmap Vulnerability Scripts
nmap --script vuln 192.168.1.20

Run Default Scripts
nmap -sC 192.168.1.20

Detect Web Applications
Metasploitable2 has several web apps.
Scan HTTP services:
nmap -p 80 --script http-enum 192.168.1.20 

SYN Scan (Stealth Scan)
sudo nmap -sS 192.168.1.20 

UDP Scan
sudo nmap -sU 192.168.1.20 

Scan Entire Network
nmap 192.168.1.0/24 

Save Scan Results
nmap -A 192.168.1.20 -oN scan_results.txt
